# Typescript 学习

## 特性

typescript 是一种给 javascript 添加特性的语言拓展，增加的功能包括：

- 类型批注和编译时类型检查
- 类型推断
- 类型擦除
- 接口
- 枚举
- Mixin
- 泛型编程
- 名字空间
- 元组
- Await

## 实现原理

typescript 是 javascript 的扩充，运行 typescript 需要先将其编译成 javascript，然后使用 node 一类的 javascript 引擎运行
